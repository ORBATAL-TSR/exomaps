# 01. Mineral Value System

- Procedural generation of mineral deposits (iron, nickel, rare earths, uranium, platinum, volatiles, etc.)
- Deposits mapped spatially: surface, subsurface, oceanic, atmospheric
- Extraction difficulty: geology, depth, planetary conditions
- Dynamic market: Prices fluctuate based on supply, demand, extraction cost, accessibility
- Resource rarity and technological breakthroughs affect value
