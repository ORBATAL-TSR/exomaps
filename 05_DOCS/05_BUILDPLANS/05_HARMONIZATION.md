# Client Harmonization & Integration

## Unified Data Model
- Shared schemas for planets, ships, economy, missions
- Real-time sync: player actions, market changes, mission status
- API or local cache for both clients

## Visual & Gameplay Consistency
- Desktop: high-fidelity 3D, advanced simulation, modding tools
- Web: accessible, lightweight, real-time dashboards, mission planning
- Shared UI/UX: mission planner, ship designer, colony manager

## Multiplayer & Cross-Client Features
- Cross-client missions, trade, science, colony management
- Modular design tools for both clients
- Advanced science panels: propulsion calculators, terraforming planners, biosphere simulators
- Harmonized visual assets: spectral colors, mineral icons, ship part models

## Improvements & Suggestions
- Move planetary/ship/economy logic to shared backend (Python/Rust microservice)
- Single source of truth for market/prices, mission status, colony state
- Modularize ship/colony design tools
- Harmonize visual and gameplay features across both clients
